Project Overview

This project involves end-to-end analysis of a pizza sales dataset using Microsoft Excel to derive actionable business insights. The objective is to evaluate sales performance, customer demand patterns, and operational efficiency through structured data cleaning, analysis, and visualization.

## Objectives

Analyze overall sales and revenue trends

Identify best- and worst-performing pizzas

Understand customer ordering behavior by time and category

Support data-driven decisions for pricing, promotions, and inventory planning

## Dataset Description

The dataset contains transactional pizza sales data with the following key attributes:

Order ID

Order Date and Time

Pizza Name and Category

Pizza Size

Quantity Ordered

Unit Price

Total Sales Value

Tools Used

Microsoft Excel

Pivot Tables

Pivot Charts

Excel Formulas

Data Validation & Filtering

Key Excel Skills Demonstrated

Data cleaning and standardization

Use of Pivot Tables for multi-dimensional analysis

Advanced formulas (SUMIFS, COUNTIFS, IF, VLOOKUP/XLOOKUP)

Time-based analysis using date and time functions

Creation of interactive dashboards using slicers

Data visualization with charts and conditional formatting

## Analysis Performed

Total revenue and total orders analysis

Sales breakdown by pizza category and size

Hourly and daily order trends

Identification of top-selling and low-performing pizzas

Revenue contribution analysis by category

Peak business hours identification

Dashboard Highlights

KPI summary (Total Revenue, Total Orders, Average Order Value)

Category-wise and size-wise sales distribution

Time-based order trend visualization

Top 5 and Bottom 5 pizzas by revenue

Interactive slicers for category and date filtering

## Key Insights

Certain pizza categories contribute disproportionately to total revenue

Medium and Large size pizzas dominate overall sales

Peak order volumes occur during specific lunch and evening hours

A small number of pizzas drive a majority of total sales

## Business Impact

Helps optimize menu offerings and pricing strategy

Supports better inventory and demand forecasting

Identifies opportunities for targeted promotions during off-peak hours


## Conclusion

This Excel project demonstrates strong foundational skills in data analysis, business reporting, and dashboard creation using Excel. It reflects real-world analytical thinking applicable to retail and food service industries.
